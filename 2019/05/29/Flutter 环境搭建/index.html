Flutter 环境搭建

1、Flutter 简述

Flutter 是谷歌的移动UI框架，可以比较快速的在Andorid 和 ios 上构建高质量的原生用户界面。Flutter基于dart语言构建，所以，原则上说，学习Flutter 应该要先掌握 Dart 语言的语法。

2、Flutter sdk 安装

下载Flutter Sdk

直接解压，然后将该路径加入 Windows 的环境变量中

同时在用户变量里新建以下条目

PUB_HOSTED_URL => https://pub.flutter-io.cn
FLUTTER_STORAGE_BASE_URL => https://storage.flutter-io.cn

3、Android-Studio 环境变量配置

flutter 的测试，打包依赖 android-studio ，因此我们也要先安装 android-studio.

下载 android-studio

直接安装即可

打开软件，找到 setting，搜索 sdk.

如图是 android-sdk 的sdk路径，和版本，根据需要可下载多个,选择后确定，安装结束后重启即可。

将 sdk 路径添加到 系统环境变量中。

ANDROID_HOME => D:\Android\sdk-pro

4、检验安装状态

在终端中输入以下命令

输出结果

flutter doctor 是 框架自带的一个检验安装状态的命令，我们可以看到我们安装了哪些东西。

这里可以看到，我们已经安装了.

  • Flutter

这样基本大功告成。

5、创建一个Flutter项目

打开终端，选定一个目录，使用命令行初始化一个Flutter示例项目。

使用 flutter create 命令，后面接自定义项目名

大功告成，一个flutter项目，就构建好了。

6、运行

在 android-studio 中，找到 avd-manage,打开虚拟机。

然后使用任意编辑器打开项目代码,推荐用 VS Code .

如上，打开项目后，点击右下角处，no-device ，可以选择已经安装的虚拟机.

点击运行
